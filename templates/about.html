The algorithm and the webapp

Recommendations were made by computing the cosine similarity between movies using user ratings, and then returning the top 10 most similar movies. The data used was from MovieLens.


This app was made using the Python web app framework Flask, and hosted on Python Anywhere


This app was a class project at CSU East Bay, dept of Statistics. It was an exercise in Python programming and recommendation algorithms.
